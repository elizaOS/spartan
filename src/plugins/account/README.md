# Account registration services for elizaOS

Giving it's own plugin incase other products or services want to build on elizaOS

Please provide feedback on how we can generalize this more and make it useful

## Actions

- I want to sign up
- Confirm email
- Delete my signup
- change account settings

## Providers

- user information
  - email address set yet or not
  - verified
- account information
  - notifications

## Services

- user
  - create user
  - update user
- account
  - create account
  - update account
