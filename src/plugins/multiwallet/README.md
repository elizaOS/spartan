# Multitenant wallet services for elizaOS

Built on account registration service.

Giving it's own plugin to create a solid foundation for creating DeFi services on elizaOS

Please provide feedback on how we can generalize this more and make it useful

## Actions

- create wallet
- import wallet
- delete wallet
- swap
- sweep
- swap all tokens to single token

## Providers

- meta-wallet information
- wallet information
- token information

## Services

- wallet services
